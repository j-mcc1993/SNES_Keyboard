SNES_Keyboard - A library for controlling keyboard output with an Arduino Leonardo or Micro and a SNES controller.

See examples for implementation.
*****************************************************************************************
*****************************************************************************************
SNES Connector layout:

	 1234 567
	——————————
	|OOOO|OOO|
	——————————

1: +5V
2: Clock
3: Latch
4: Data
5: Nothing
6: Nothing
7: Ground
